#Ticketing-system backend

##Features
-User Authentication
-Create Ticket
-Assign Ticket
-View Tickets

##Tech Stack
-Java 17
-Spring Boot
-MySQL
-Spring Data JPA
-Maven

##How to run
1. Open project in IntelliJ
2. Configure database in application.properties
3. Run TicketingBackendApplication.java
4. Open http://localhost:8080
